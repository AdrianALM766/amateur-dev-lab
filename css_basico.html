CSS

Que es css y para que sirve

CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para estilizar documentos HTML y XML. Su función principal es controlar la presentación y el diseño
de una página web, permitiendo cambiar el aspecto visual de los elementos HTML, como colores, fuentes, márgenes, tamaños y posicionamiento. En resumen, CSS se utiliza
para dar estilo y mejorar la apariencia de las páginas web.
