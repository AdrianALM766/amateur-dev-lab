Formularios

Que son?

Los formularios son elementos que permiten a los usuarios enviar datos al servidor web para su procesamiento.
Son una parte fundamental de las páginas web interactivas y se utilizan para recopilar información
del usuario,como nombres, contraseñas,comentarios, opciones seleccionadas, etc. Para crear un formulario en HTML,
se utiliza el elemento form

input

  • se utiliza para crear diversos tipos de campos de entrada en formularios, como cajas de texto, botones,
    casillas de verificación, botones de radio, etc.
  • Se define mediante la etiqueta input con un atributo type que especifica el tipo de entrada que se desea crear
    por ejemplo, text para cajas de texto, checkbox para casillas de verificación (mas adelante se explica), etc.

label

  • se utiliza para asociar un texto descriptivo con un elemento de formulario, como un input
  • Se asocia con un elemento de formulario mediante el atributo for, que tiene el mismo valor que
    el atributo id del elemento de formulario al que está etiquetando

Curiosidades

Checkbox

Para hacer que un checkbox o un radio button este seleccionado por defecto, necesitas añadir el atributo checked. No hay necesidad de agregarle un
valor al atributo checked. Simplemente necesitas añadir la palabra checked al elemento input, asegurándote de que hay espacios entre ella y los otros atributos.

action

action sirve para decir a donde se mandara los datos recolectados de formulario

post

En HTML, el método HTTP POST se utiliza dentro de un formulario (

) para enviar datos al servidor. Cuando un usuario completa un formulario en una página web y
hace clic en el botón de envío, los datos del formulario se envían al servidor utilizando el método POST.
